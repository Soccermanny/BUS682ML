BUS682 Project 1 - Runbook (All Data + Processes)
Date: 2026-03-31

Overview
This folder contains three main pipelines:
1) IMDb scrape -> output/imdb_ratings_output.xlsx (in GCS)
2) Rotten Tomatoes scrape -> output/rotten_tomatoes_scores.xlsx (in GCS)
3) Movie clustering (TMDB + Wikipedia + Wikidata) -> movie_clusters.csv + movie_clusters.html

Main project scripts
- scraper_imdb.py        (IMDb scraper; GCS input/output)
- main.py                (another IMDb scraper variant; similar to scraper_imdb.py)
- scraper_rottentomatoes.py (RT scraper; reads IMDb output from GCS)
- movie_clustering.py    (clustering/enrichment; local or GCS)
- cluster_features.py    (feature taxonomy clustering from CSVs)

Prerequisites
1) Python 3.11+ (project currently uses a local .venv)
2) pip packages
   - For scrapers:
     pip install -r requirements.txt
   - For clustering:
     pip install -r requirements.clustering.txt
3) If using GCS/Cloud Run:
   - Install gcloud CLI and authenticate
   - Have access to the bucket and Cloud Run jobs

Windows local setup
1) Open PowerShell in this folder
2) Activate venv
   .\.venv\Scripts\Activate.ps1
3) Install dependencies (if needed)
   pip install -r requirements.txt
   pip install -r requirements.clustering.txt

Pipeline A - IMDb scrape (GCS based)
Script: scraper_imdb.py (or main.py)

Required environment variables:
- GCS_BUCKET_NAME
- INPUT_FILE          (default: input/movies.csv)
- OUTPUT_FILE         (default: output/imdb_ratings_output.xlsx)
- MAX_WORKERS         (default in script: 5 or 10 depending on file)

PowerShell example:
  $env:GCS_BUCKET_NAME = "your-bucket"
  $env:INPUT_FILE = "input/movies.csv"
  $env:OUTPUT_FILE = "output/imdb_ratings_output.xlsx"
  $env:MAX_WORKERS = "5"
  python .\scraper_imdb.py

Pipeline B - Rotten Tomatoes scrape (GCS based)
Script: scraper_rottentomatoes.py

Required environment variables:
- GCS_BUCKET_NAME
- INPUT_FILE          (default: output/imdb_ratings_output.xlsx)
- OUTPUT_FILE         (default: output/rotten_tomatoes_scores.xlsx)
- MAX_WORKERS         (default: 10)

PowerShell example:
  $env:GCS_BUCKET_NAME = "your-bucket"

  $env:OUTPUT_FILE = "output/rotten_tomatoes_scores.xlsx"
  $env:MAX_WORKERS = "10"

Pipeline C - Movie clustering + enrichment
Script: movie_clustering.py

Required for local run:
- TMDB_API_KEY        (required)
- MOVIES_CSV          (default: genre_data.csv)

Optional for Cloud Run/GCS mode:
- GCS_BUCKET_NAME
- INPUT_BLOB          (default: output/imdb_ratings_output.xlsx)
- OUTPUT_PREFIX       (default: output/clustering)

PowerShell local example:
  $env:TMDB_API_KEY = "your_tmdb_key_here"
  $env:MOVIES_CSV = "genre_data.csv"
  python .\movie_clustering.py

Expected local outputs:
- movie_clusters.csv
- movie_clusters.html
- cache_tmdb.json, cache_wikipedia.json, cache_wikidata.json, cache_translations.json
